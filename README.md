# Proyecto Tarjeta Interactiva

Proyecto web de invitación interactiva con contador, música y sección de regalos.

## Tecnologías
- Angular
- TypeScript
- Tailwind CSS
- HTML / CSS
- Audio con HTMLAudioElement

## Funcionalidades
- Contador regresivo de días, horas, minutos y segundos.
- Reproducción de música de fondo.
- Botón para agendar evento en el calendario.
- Modal con datos de regalo.
- Secciones interactivas con animaciones y SVG.
